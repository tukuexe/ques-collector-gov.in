About This Project

As a Class 10 student myself, I understand the struggle of finding quality practice papers. That's why I built HSLC Paper Vault - to create a centralized repository where students from all over Assam can contribute and access question papers from different schools and districts.

This platform is built with security and privacy in mind. No credentials are stored, and all uploads are anonymous by default. The papers go directly to a private Telegram bot where they're organized and shared with the student community.
